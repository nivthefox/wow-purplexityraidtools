PRT Notes — In-Game Test Plan

rev 4 (list + modal UI) · branch feature/notes · headless suite: 243 passed, 0 failed

How this works. Everything is done through the UI; the only slash command is /reload. Note text goes into the Edit modal (the tab itself never shows note text). Send is how a note becomes active — solo, Send sends to yourself. Show/Hide previews the selected note without activating anything. Internals unreachable from the UI (parser edges, gate ordering, invalid-payload rejection, tick math, solo-send wiring) are pinned by the 243-test automated suite.
Sample notes — paste into the modal's text field as directed.
